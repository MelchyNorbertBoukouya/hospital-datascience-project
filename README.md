# 🏥 Hospital Data Science Project

## 📌 Overview

This project analyzes hospital performance and predicts stockout risks using data science techniques.

## 🎯 Objectives

* Analyze hospital activity
* Identify critical situations
* Predict stock shortages

## 📊 Dataset

The dataset includes:

* Number of patients
* Medical stock levels
* Staff availability
* Power status
* Risk of stockout

## 🧪 Methodology

1. Data Cleaning
2. Exploratory Data Analysis (EDA)
3. Feature Engineering
4. Machine Learning (Random Forest)
5. Evaluation (Accuracy, Precision, Recall, F1-score)

## 📈 Results

* Identification of high-risk situations
* Key factors influencing stock shortages
* Predictive model for hospital risk

## 📊 Dashboard

An interactive Power BI dashboard was created to visualize:

* Patient trends
* Stock levels
* Risk indicators

## 🛠️ Tools

* Python (Pandas, Scikit-learn)
* Power BI
